MLBeacon

TECH SPEC

Size and Weight

● Height: 64 mm
● Width: 64 mm
● Depth: 18 mm
● Weight:50 g

Power Supply

● AA Battery x 2
● Battery life: 2 years

Barcode Reader

● Bluetooth V4.0 LE compliant
● Profiles Supported: Proximity
● Wireless Range: 10~30 m / 33~100 feet
● Transmit output Power: Class II Device -6≦Output Power≦+4
● RF Frequency Range: 2.402~2.480 GHz
● Receiver Sensitivity: <0.1% BER at -70 Dbm
● Operating Voltage: 3.3V + 0.3V / 3.3V - 0.3V

Operatin Temperature and Humidity

● Operating Temperature: 0 °C to 50 °C (32 °F to 123 °F)
● Relative Humidity: 5 - 90% (non-codesnsing)

Toolkit List

● iOS:
1. Sample code for basic testing.
2. Toolkit for changing Major, Minor, RSSI.
3. Full SDK that can detect current battery capacity, change all the data listed above, and also a version with a setting for higher sensitivity.

● Android:
1. Sample code for basic testing.
2. Open source SDK that can change UUID and related data
